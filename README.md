## Структура проекта
index.html — главная страница и структура приложения.

script.js — логика инициализации карты, база данных на 31 организацию и алгоритм фильтрации.

style.css — мультяшные стили, анимации и оформление карточек.

package.json — конфигурационный файл проекта с зависимостями.

##  Инструкция к запуску кода
npm install
npm start
